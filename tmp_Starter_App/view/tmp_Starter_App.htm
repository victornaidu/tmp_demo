About tmp_Starter_App

This demo application is assembled using Trillo builder. Trillo follows modular approach - build components and stitch them together as an application. The components are metadata driven and thus allow customization.

The componentized approach of Trillo makes development of an enterprise class UI efficient, flexible and modular. Persons with different roles can use its visual or programming environment to contribute to the end product.

Trillo transparently uses Git to enable collaboration among team members.
